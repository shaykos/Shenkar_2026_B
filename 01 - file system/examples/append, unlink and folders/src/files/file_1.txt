The Old Library

Sarah discovered the letter tucked behind a loose brick in the library's foundation. 
It was dated 1952, written in faded ink by someone named Eleanor.

"If you're reading this," it began, "then the library still stands. That brings me joy."

Sarah smiled and carefully placed the letter back, adding her own note beneath it for 
whoever might find it fifty years from now.

The library would endure. Stories always do.
